# ChatKit News Guide Demo

An AI-powered news assistant demo built with OpenAI ChatKit.

## Features

- Article search and recommendations
- @-mentions for articles and authors
- Page-aware contextual responses
- Entity tag previews
- Streaming AI responses

## Running on Replit

1. Import this repo on Replit
2. Add your `OPENAI_API_KEY` in Secrets
3. Click Run

## Local Development

```bash
# Set your API key
export OPENAI_API_KEY="your-key"

# Run the app
bash run.sh
```
